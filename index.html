📋 使用说明

  1. 此工具基于您提供的Kernel类验证逻辑，生成对应的SN码
  2. SN码生成算法与您的系统完全一致：
    - 域名SN码: strtoupper(substr(md5(substr(sha1(domain), 0, 10)), 10, 10))
    - IP SN码: strtoupper(substr(md5(substr(sha1(ip), 0, 15)), 10, 10))
    - 用户SN码: strtoupper(substr(md5(substr(sha1(user), 0, 20)), 10, 10))
  3. 新功能：为域名生成两个SN码
    - 为输入的域名（如：example.com）生成一个SN码
    - 为带"www."前缀的域名（如：www.example.com）生成另一个SN码
    - 如果输入的是带www.的域名，也会为不带www.的版本生成SN码
  4. 生成的SN码是10位大写字母数字组合
  5. 您可以将多个SN码合并到一个数组中，系统会自动去重
  6. 表单中的"已有SN码"字段可以合并您之前生成的SN码
  7. 注意：实际生成逻辑在安全的后端API中，不会泄露算法
